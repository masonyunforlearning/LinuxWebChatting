실행 파일 이름 : client
실행 : ./client <server ip> <port>

처음 입장시 자신의 닉네임 입력

그 후 로비에 입장

로비에서는 메세지를 보낼 수 없고 일부 명령어만 사용가능
방 안에서만 방안의 멤버들과 채팅 가능

로비에서 사용가능한 명령어
/enter <방 번호> : 해당 방에 입장합니다.
/mkroom <방이름> : 해당 이름의 방을 만듭니다. 

방에서만 사용가능한 명령어
/exit : 현재 속한 방에서 로비로 나갑니다.
/info : 현재 속한 방의 정보(방의 이름, 번호, 멤버 등)을 보여줍니다.

어디서든 사용가능한 명령어
/myname : 자신의 이름을 보여줍니다.
/quit       : client를 종료합니다.
/help, /?  : 사용가능한 모든 명령어와 설명을 보여줍니다.
/dm <대상> <메세지> : 대상에게만 메세지를 보냅니다.
/rlist : 방의 목록들을 보여줍니다.
/mlist : 접속한 사람들을 보여줍니다.
